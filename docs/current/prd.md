# AI 학습 기록 블로그 서비스 — 사업 제안서 (PRD)

────────────────────────────────
1. 사업 개요
────────────────────────────────

AI를 공부하는 사람들이  
자신의 학습 과정을 기록하고,  
그 기록을 기반으로 인사이트를 얻을 수 있는  
개인용 기록 서비스입니다.

단순 블로그가 아니라  
"기록을 쌓으면 자동으로 정리가 되는 도구"입니다.


────────────────────────────────
2. 왜 필요한가 (문제 정의)
────────────────────────────────

현재 AI를 공부하는 사람들은 다음과 같은 문제를 겪고 있습니다.

- 배운 내용을 기록하지 않는다 (귀찮아서)
- 기록을 해도 다시 보지 않는다
- 정리가 안 되어 실력으로 이어지지 않는다

결론적으로,

많이 배우지만  
남는 것이 없는 상태가 반복됩니다.


────────────────────────────────
3. 해결 방식
────────────────────────────────

이 서비스는  
“기록 → 자동 정리 → 인사이트 제공” 흐름을 만듭니다.

사용자는 단순히  
배운 내용을 짧게 적기만 하면 됩니다.

이후 시스템이 자동으로:

- 핵심 내용 요약
- 주요 키워드 정리
- 다음 학습 방향 제안

까지 제공합니다.

즉,

"생각하지 않아도 정리되는 환경"을 제공합니다.


────────────────────────────────
4. 서비스 구조 (매우 단순)
────────────────────────────────

전체 서비스는 3개의 화면으로 구성됩니다.

1. 기록 페이지
- 배운 내용을 간단히 작성
- 이전 기록 확인

2. 상세 페이지
- 작성한 내용이 자동 정리된 결과 확인

3. 인사이트 페이지
- 누적된 기록을 기반으로 학습 패턴 분석

복잡한 기능 없이  
“기록 → 확인 → 성장” 흐름만 제공합니다.


────────────────────────────────
5. 기대 효과
────────────────────────────────

사용자 관점:

- 기록 부담 감소 → 기록량 증가
- 학습 내용 자동 정리 → 이해도 상승
- 반복 학습 구조 형성 → 실력 축적

사업 관점:

- AI 학습자 대상 강력한 락인 서비스
- 교육 / 커뮤니티 / 콘텐츠 서비스로 확장 가능
- 개인 데이터 기반 고부가가치 서비스 전환 가능


────────────────────────────────
6. 차별점
────────────────────────────────

기존 블로그:
→ 글을 잘 써야 한다

이 서비스:
→ 그냥 쓰면 자동으로 정리된다

핵심 차이는 하나입니다.

"기록의 부담을 제거한다"


────────────────────────────────
7. 실행 전략 (초기)
────────────────────────────────

- 최소 기능으로 빠르게 출시
- 개인 사용 중심으로 검증
- 이후 커뮤니티 / 공유 기능 확장

핵심은:

"잘 만든 서비스"가 아니라  
"매일 쓰게 만드는 서비스"입니다.


────────────────────────────────
8. 결론
────────────────────────────────

이 서비스는 단순한 기록 도구가 아니라,

AI 학습 과정에서  
"지식이 축적되는 구조"를 만드는 제품입니다.

작게 시작하지만,  
사용자의 데이터를 기반으로  
확장성이 매우 높은 서비스입니다.

→ 지금 시작하는 것이 적절합니다.
