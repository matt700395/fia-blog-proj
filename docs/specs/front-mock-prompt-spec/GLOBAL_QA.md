# GLOBAL_QA

────────────────────────────────────────────────────────
[목적]
────────────────────────────────────────────────────────

이 문서는 프론트엔드 목업의 완료 기준과 검증 항목을 정의한다.

모든 프론트 목업은 이 QA 기준을 통과해야 "완결 상태"로 인정된다.

---

────────────────────────────────────────────────────────
[핵심 원칙]
────────────────────────────────────────────────────────

- 모든 라우터는 독립적으로 동작해야 한다
- 모든 상태는 반드시 확인 가능해야 한다
- 사용자 흐름은 끊기지 않아야 한다
- UI는 즉각적인 피드백을 제공해야 한다

---

────────────────────────────────────────────────────────
[1. 라우터 검증]
────────────────────────────────────────────────────────

□ 모든 라우터 직접 접근 가능  
□ 라우터 이동 정상 작동  
□ URL 파라미터 정상 처리  
□ 브라우저 뒤로가기/앞으로가기 정상  

---

────────────────────────────────────────────────────────
[2. JSON 렌더링 검증]
────────────────────────────────────────────────────────

□ 모든 페이지가 JSON 기반으로 렌더링됨  
□ 데이터가 없을 경우 empty 상태 정상 표시  
□ JSON 변경 시 UI 정상 반영  

---

────────────────────────────────────────────────────────
[3. 상태 전이 검증]
────────────────────────────────────────────────────────

각 라우터마다 반드시 확인:

□ initial 상태  
□ loading 상태  
□ success 상태  
□ empty 상태  
□ error 상태  

---

────────────────────────────────────────────────────────
[4. 사용자 인터랙션 검증]
────────────────────────────────────────────────────────

□ 모든 버튼 클릭 시 반응 있음  
□ 클릭 → UI 변화 발생  
□ 이벤트 누락 없음  

---

────────────────────────────────────────────────────────
[5. 네비게이션 검증]
────────────────────────────────────────────────────────

□ 주요 흐름 3개 이상 정상 동작  
□ 라우터 간 이동 자연스러움  
□ dead-end 페이지 없음  

---

────────────────────────────────────────────────────────
[6. 입력/폼 검증]
────────────────────────────────────────────────────────

□ 입력 시 값 반영  
□ validation 정상 동작  
□ 에러 메시지 표시  

---

────────────────────────────────────────────────────────
[7. UI 상태 검증]
────────────────────────────────────────────────────────

□ loading UI 표시됨  
□ empty 상태 안내 존재  
□ error 상태 안내 존재  
□ 재시도 가능  

---

────────────────────────────────────────────────────────
[8. 일관성 검증]
────────────────────────────────────────────────────────

□ 버튼 스타일 일관성  
□ 레이아웃 일관성  
□ 인터랙션 패턴 일관성  

---

────────────────────────────────────────────────────────
[9. 성능/UX 검증]
────────────────────────────────────────────────────────

□ 클릭 후 즉시 반응  
□ 로딩 상태 표시  
□ UX 끊김 없음  

---

────────────────────────────────────────────────────────
[완료 기준]
────────────────────────────────────────────────────────

아래 조건을 모두 만족해야 한다:

- 모든 라우터 정상 작동
- 모든 상태 전이 확인 완료
- 사용자 흐름 완성
- UI 피드백 정상

---

────────────────────────────────────────────────────────
[AI 수행 규칙]
────────────────────────────────────────────────────────

- 모든 프론트 목업은 이 QA 기준을 통과해야 한다
- QA 항목이 하나라도 실패하면 "완결"이 아니다
- QA 실패 시 해당 라우터를 수정한다
